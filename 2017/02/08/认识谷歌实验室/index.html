谷歌实验简介

发表于 2017-02-08 | 分类于 -Blog |

这是我的第一篇博文，既然是写第一篇，那就得写一些不同凡响的东西–谷歌实验。

Chrome Experiments 是Google在2009年推出的计划，它是一个创造性的网站试验的陈列柜，其中大部分的产品都是最近才开发出来的新技术，包括HTML5, Canvas, SVG, 和WebGL。所有的这些产品都是由那些来自世界各地的有才华的艺术家和程序员们创建和提交的。它向世人展示到底 HTML5 和 JavaScript 技术能创造什么美丽或新奇的网路体验。

下面这些是我从Chrome Experiments收集的一些网站，有兴趣的呃可以去看一下。

说实话，下面这个是我收藏中最喜欢的一个网站之一，也是我见过最装逼的“相册”，希望以后有一天能把这个网站的源代码copy过来。哈哈哈～

  • http://www.hiroshi-yoshida.jp/imago/

这个网站绝对颠覆你对mv的认知，个人觉得应该算是现代音乐创作的最前沿吧。

  • http://film.livyatanim.com/film.html#

这是一个父亲为他患白血病的儿子所创作的一个3D版的mv，看完满满的感动。（后来听说他儿子的病治好了）

  • https://throughthedark.withgoogle.com/

很奇妙的一次音乐之旅，是一场视觉，听觉的盛宴。可惜看不懂德文。

  • http://cabbi.bo/enough-deutsch/

无聊或者心情烦躁的的时候可以打开这个网站，听听雨的声音。

  • http://www.rainymood.com/

这是一个很简单的网页游戏，但很有趣也很好玩。（表示已经通关好多次了）

  • https://rotopo.com/

想感受被面食包围的恐惧吗？我想这个网站很适合你。

  • http://alteredqualia.com/three/examples/webgl_pasta.html

不知道你们有没有想过这样一个情景：“在夜黑风高的一个夜晚，你孤身一人在一个未知的森林里被不知名的东西一直追着，一直…”

  • http://oos.moxiecode.com/js_webgl/forest/index.html

想知道用电脑制作出来的“大树”是怎么样的吗，这个网站可以满足你的好奇心。

  • http://www.bongiovi.tw/experiments/webgl/blossom/

随便的在浏览器上画一段线段，然后他就会为你匹配出某个图片的某个部分出来，感觉蛮好玩的。

  • https://lines.chromeexperiments.com/

一只动态的3D水母。说实话，我从来没想过一只水母可以美成这样。

  • https://jayweeks.com/medusae/

珍珠男孩—一个简单3D网页游戏，你可以在那里划划船，看看日落什么的,不喜欢的话还可以跳到海里面去，去看看海底的世界是怎么样的。

  • http://labs.gooengine.com/pearl-boy/?remote=true&id=3200
