《大数据时代》笔记

发表于 2017-03-02 |

前几天看了一下迈尔和库克写的《大数据时代》。在看的过程中，对大数据有了一些新的认识，于是把它整理成了这篇笔记。

首先我们要认识什么是大数据。大数据是指采用所有的信息来做分析，并非单单指数据量的庞大，虽然大多数情况下的确是如此。大数据有三个核心的理念。第一是做数据分析时，我们要的是所有的数据而不是抽样数据。第二是我们对大数据的处理结果要的是效率而不是精确程度，大数据从来不追求对某一事件的精确判断，他要的是方向，是事物发展的整体趋势。第三是大数据处理需要的是“相关关系”而不是“因果关系”，也就是说大数据能让我们知道是什么而不是为什么。例如大数据会告诉我们把避孕套，口香糖放在超市门口会更容易卖出去，但他不会告诉我们为什么会这样。简而言之，大数据就是利用全部的信息来挖掘其中的相关关系，虽然有时这些关系听起来不是那么的靠谱。

以前没有大数据是因为收集数据本身就是一件不容易干的事，而且也没有足够的计算能力来处理庞大的数据。然而今天不同了，随着存储设备的大幅度降价和微型传感器的广泛使用，尤其是在手机等各种智能设备的使用下，各式各样的微型传感器遍布我们的生活之中，这使的收集数据变的异常容易起来。其次是计算机的计算能力有了长足的进步，也使的在短时间内处理庞大的数据成为了可能。

随着行业领域的不断划分，小数据时代的抽样处理数据会变的异常困难起来，有时候还容易得出错误的结论。例如在1936年，《文学摘要》对罗斯福和兰登谁能当选美国总统的预测，就是抽样数据不准而导致的失误。但是大数据不会，而且由于使用了所有的数据，大数据相对于抽样数据还能观察到更加细致不易察觉到的东西，这更加有利于把握事物的发展方向，从而做出一些正确的选择。

但是大数据也有一些不好的地方。首先由于大数据使用的事所有的数据，那么这些数据中有一些是有效数据，有一些是无效数据，这就牺牲了结果的精确性。也就是说，大数据只能给出整体的大致情况，而在面对具体某个事物的时候，他是无法给出预测结果的。大数据以牺牲精确性换取方向性，这有点像宏观物理和量子力学的区别。

其次是大数据会让我们的隐私无所遁形，使我们变的毫无秘密而言。相关实验表明，即使你在网上所有数据是匿名的，但只要将你的数据进行一定的配对处理，依旧可以把你的真实身份找出来，换一句话就是说就是在新的时代中你几乎是透明的。而更加可怕的不是个人隐私的泄漏，而是被预知的可能性。由于人可以被预知，那么个人在贷款，工作等各方面生活中必将受到各种不公平的待遇，这对于讲究平等的我们来说简直就是个灾难。

最后，当数据规模到达一定程度之后，继续扩充的意义已经不大，更何况所有大数据的应用都极大的依赖于其选取的数学处理模型。而经济发展的不均衡决定了所谓的样本=总体永远不可能出现——全世界有十几亿人还处日生活开销不足2美元的贫困线以下，连登录网络的条件都不具备，我们怎么能够奢望我们收集到的数据就会是总体？所以我认为在应用层面上而言，大数据有其可用之处，但对我们如何更好的了解这个世界的真相，却毫无用处。

# -大数据 data notes Viktor Mayer-Schönberger
