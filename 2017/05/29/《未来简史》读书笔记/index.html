《未来简史》读书笔记

发表于 2017-05-29 | 分类于 -Notes | | 阅读次数

人类的新议题

旧议题：

饥荒，瘟疫，战争

新议题：

我们应该把注意力和创造力放在哪里？
手上掌握力量的我们，究竟该如何利用？

笔记：

历史的作用：研读历史是为了让我们在未来的道路上知道更多的选择。
人类之所以不愿意改变，是因为害怕未知。但历史唯一不变的，就是一切都会改变。
心灵是人工智能的最高等级。
永恒的意识和灵魂是由具体的行动体现出来的。
真正的能力在于做出正确的选择，以及把握好时机。
在获得某个机会时，如何正确的使用才是最关键的。
如果人类没有想象，那和人工智能有什么区别？
做一个现实的理想主义者。
随着社会的发展，人类的生存环境会越来越恶劣，这不仅仅是说生理上的，还有心理上的问题。
人不利己，天诛地灭，自己过的好才是真的好，不要扯那些虚构的荣誉和团体。
神从未消失，在新的时代，人类用更强大的科技技术创造出更强大的符合人类幻想的神。虚拟与现实的界限变的越来越模糊。
人类要合作，就不能只有问题，而需要坚定的答案，这才是他们追随你的原因。
人类能从自然竞争中胜出的原因在于合作。
我们与前人最大的不同在于，我们相信未来。
科学的力量越大，带来的威胁也越大。
现代社会以增长设为成功的目标，并将稳定视为洪水猛兽。
人文主义时期，人们更在乎体验和感受，而不是做了什么行为。
在现代社会，想要生存，就不仅需要了解丛林法则，还要学会开心的拥抱丛林法则。
活在幻想永远都会是一个轻松的选项，他让你所受的一切痛苦都有了意义。（事实并非如此）
未来是算法的，是设计师的，无论什么职业也是如此。
如果贫富差异不仅仅体现在财富上，还体现在基因上，那这个世界会变成什么样子呢？
未来的世界永远是精英的世界，这点自古以来从未改变。
“数据主义”是当代的新兴宗教。
